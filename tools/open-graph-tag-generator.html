What Open Graph tags do

Open Graph tags control how a page can appear when shared on social platforms or messaging apps. A good preview usually includes a title, description, URL and image.

These tags do not replace SEO title and description tags, but they support better presentation when people share your content. Use accurate titles and images that match the page.

Useful fields

  • og:title for the shared headline.
  • og:description for the short preview text.
  • og:url for the canonical page URL.
  • og:image for the preview image.

After adding tags

Test the page with social preview tools when available. Make sure the image loads over HTTPS and the preview text is not misleading.

Are Open Graph tags required for SEO?

They are not required for ranking, but they improve how pages appear when shared.

Yes. In most cases it should point to the preferred canonical URL.
