Why schema markup is useful

Schema markup is structured data that gives search engines additional context about a page. It can describe an organization, website, article or other content type in a machine-readable way.

This tool creates basic JSON-LD for common page types. It is useful for simple pages, but you should always review the output and make sure it matches the visible content.

Before publishing schema

  • Choose the schema type that matches the page.
  • Use accurate names, URLs and descriptions.
  • Do not mark up content that is not on the page.
  • Test the final page after deployment.
