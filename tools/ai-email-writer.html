FAQs

Is AI Email Writer free?

Yes. It is free to use on ToolPinch, subject to fair-use limits and API availability.

Does this expose the API key?

No. The API key is stored in Vercel environment variables and used only by the serverless API route.

Can it send the email for me?

No. It only creates a draft that you can copy, review and send yourself.

Should I include private details?

Avoid sensitive information. Use placeholders and add private details only in your email app if needed.
