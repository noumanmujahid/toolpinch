Privacy Policy

ToolPinch is designed to keep common tools simple and privacy friendly. The current calculators, text tools and password generator run in your browser and do not require an account.

Information you enter

For the current static version, tool inputs are processed locally in the browser. Do not enter highly sensitive information into any online tool unless you understand how that page works.

Analytics and ads

If analytics or advertising is added in the future, this page should be updated with the provider names, cookie usage and opt-out information. Add only approved ad code and follow the ad network's policies.
