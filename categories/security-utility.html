Security and utility tools for safer everyday tasks

This category includes a password generator, password strength checker and QR code generator. The password tools run in the browser and are designed for quick checks without requiring an account.

Generated passwords should be stored in a trusted password manager and never reused across important accounts. A strength estimate is guidance only; account security also depends on uniqueness, multi-factor authentication and protection against phishing.

Useful safety habits

  • Use a different password for every important account.
  • Choose long passwords generated with secure browser randomness.
  • Turn on multi-factor authentication where available.
  • Test QR destinations before printing or sharing them.
  • Never place private information in a public QR code.
