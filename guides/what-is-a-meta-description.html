What Is a Meta Description?

A meta description is a short summary of a web page. Search engines may use it as the snippet below a page title in search results. A good description helps people understand what the page offers before they click.

Meta descriptions are one part of a larger SEO workflow, and they can improve click-through rate when they match search intent. A clear snippet can make your page look more useful than competing results.

What to include

Write one natural sentence or two short sentences. Mention the main topic, the benefit for the visitor and any important detail that makes the page useful. Avoid repeating the same description across many pages.

Good meta description habits

  • Keep the description specific to the page.
  • Use the main phrase naturally, not repeatedly.
  • Make the benefit clear without exaggerated claims.
  • Preview the snippet before publishing.
