What Is FAQ Schema?

FAQ schema is structured data that describes questions and answers on a page. It remains part of schema.org vocabulary when it accurately matches visible content.

Google stopped displaying FAQ rich results in May 2026 and removed the feature documentation in June 2026. FAQ markup should therefore not be added with the expectation of receiving an expanded Google search result.

When to use it

Use FAQPage markup only when another platform or an internal data workflow needs it and the visible page contains the same questions and answers. Helpful visible FAQs can still support visitors without structured data.

Basic checklist

  • Show every marked-up question and answer on the page.
  • Use clear, direct and accurate answers.
  • Do not promise Google FAQ rich results.
  • Validate JSON-LD before publishing it.
