How to Write Meta Descriptions

A meta description is a short summary of a page. It should tell visitors what the page offers and why it is useful. Search engines may rewrite snippets, but a clear description is still worth writing.

For tool pages, mention what the visitor can do. For guides, mention what the visitor will learn. Keep the description specific and avoid using the same text across many pages.

Useful formula

Use this page to do [task], understand [topic] or create [result]. Simple, free and easy to use in your browser.

Common mistakes

  • Writing vague descriptions.
  • Repeating keywords unnaturally.
  • Making claims the page does not support.
  • Using the same description for every page.
